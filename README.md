# 数之谜数学供题库（修复版）

## 最快使用

1. 解压 `数之谜_GitHubPages网站_修复版.zip`。
2. 将压缩包内第一层的全部文件上传到 GitHub 仓库根目录。
3. 在 Supabase SQL Editor 运行 `supabase/schema.sql`。
4. 在 Authentication → Users 创建并自动确认 `adminpzh@math.example`。
5. 如登录异常，运行 `supabase/管理员检查与修复.sql`，并打开 `管理员登录修复.html`。
6. 登录后导入 `setup/首次导入题库_42题.zip`。

本版已修复 SVG 图标异常、手机/平板适配、配置文件不一致和登录排错提示。
