# 数之谜数学供题库：混合格式修复版（r4）

本版本针对线上反馈继续修复：

1. 重写数学内容解析器，不再依赖容易截断公式的单层正则表达式。
2. 同时兼容标准 LaTeX、Markdown 和普通文本式数学标记。
3. 支持独立一行的 `[` / `]` 作为展示公式边界。
4. 支持 `(n=1)`、`(n\ge 2)`、`(L(w))` 等圆括号式行内公式输入。
5. 支持 `###` 标题、`---` 分隔线、项目符号和粗体。
6. 后台新增“预览格式”，保存前可直接检查题目和解答。
7. 仅保留原始 TeX 中确实存在 TikZ 的 PZH-009、PZH-033 两张图。
8. 四大模块改为唯一主分类；42 道题已人工重新归类。

部署与更新步骤见 `使用方法.txt` 或 `本次更新说明.html`。


## 2026-07-20 r6 公式排版修复

- 自动修复常见的 `\left` / `\right` 缺失、错误花括号和不配对问题。
- 长公式先按容器宽度缩小，仍过长时仅公式区域横向滚动，不再撑出屏幕。
- 独立的短公式会自动并排并换行，减少不必要的大段留白。
- 手动新增、TeX 导入和 ZIP 批量导入统一使用同一套内容清洗与排版流程。
- TeX 手动导入后自动打开整理后的显示预览。
