日期：2026-07-20

本版修复：
1. 修复 MathJax “Missing or unrecognized delimiter for \\left” 类错误。
   - 自动识别真正的 \\left / \\right 命令，不误伤 \\leftrightarrow。
   - 自动修复 \\left{、\\right} 等常见花括号写法。
   - 遇到不配对或无法识别的自动伸缩分隔符时，保留括号并安全降级显示。

2. 长公式不再撑出手机或电脑屏幕。
   - 优先按容器宽度适度缩小。
   - 极长公式只在公式区域内横向滚动，不影响整页宽度。
   - 窗口尺寸变化时自动重新计算。

3. 短公式智能紧凑排版。
   - 独立且较短的公式自动并排。
   - 空间不足时自动换行，减少每个公式单独占一整行造成的留白。

4. 手动导入与批量导入统一美化。
   - 手动新增、从 TeX 识别、ZIP 批量导入共用同一套清洗和公式修复流程。
   - 从 TeX 导入后自动“整理并预览”。
   - 保存前再次统一格式，避免手动导入和原题库显示效果不同。

部署：
将本压缩包全部文件覆盖到原 GitHub Pages 仓库并提交即可。
